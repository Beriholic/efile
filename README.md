# efile
A cli tool for simple encryption of files

## use
encrypt
```bash
efile enc <file1> <file2> ... <fileN>  <key>

efile enc <file_folder> <key>
```

decrypt
```bash
efile dec <file1> <file2> ... <fileN>  <key>

efile dec <file_folder> <key>
```
## install
```bash
go install github.com/Beriholic/efile@latest
```
